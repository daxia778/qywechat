# 更新日志 (Changelog)

## v1.6.2.3 — 2026-04-12

### 🐛 Bug 修复

- **收款流水筛选修正**：收款流水页仅显示已完成（COMPLETED）订单的收款记录，排除待处理/设计中/退款订单的流水
- **退款数据同步**：修复设计师花名册页退款订单数始终显示为 0 的问题（前端字段名 `refund_orders` → `refunded_orders`）
- **展开详情定位**：修复收款流水页「关联订单」展开后详情卡片跑到列表最底部的问题，现在紧贴对应行下方

### ✨ 功能优化

- **设计师搜索增强**：支持从企微团队通讯录 + 外部联系人 + 花名册三重范围搜索设计师
- **智能推荐**：不输入关键词时自动推荐花名册中综合评分最高的 3 位设计师
- **搜索性能**：服务启动 2 秒内自动预热企微通讯录内存缓存，首次搜索延迟从 ~400ms 降至 <5ms
- **花名册实时同步**：接入 WebSocket `order_updated` 事件监听，订单状态变更后花名册数据自动刷新
- **展开详情信息丰富化**：订单详情卡片新增页数、备注字段展示

### 📦 技术改进

- 新增 `wecom_contact_cache.go`：企微通讯录内存缓存（10 分钟刷新），含团队成员 + 外部联系人
- 新增 `wecom_external_contact.go`：外部联系人数据模型
- 收款汇总统计同步应用 COMPLETED 过滤规则
